# HR-Attrition-Analysis-Dashboard
Overview
The HR Attrition Analysis Dashboard is a data-driven project focused on analyzing employee attrition using structured HR data. The solution involves data extraction, cleaning, and transformation using Power Query, followed by data modeling and visualization in Excel. The dashboard is designed to uncover attrition trends across multiple dimensions such as department, job role, tenure, and demographics, enabling deeper workforce analytics.

Key Metrics
The analysis covers 1,470 employees with an overall attrition rate of 8%. Additional KPIs include overtime (28%), average salary hike (15%), average tenure (7 years), and average monthly salary of ₹6,503, providing a quantitative foundation for workforce evaluation.

Features
The dashboard incorporates interactive slicers and dynamic visuals to support multi-dimensional analysis. It includes tree maps for job role distribution, bar charts for departmental analysis, and trend visualizations for tenure-based attrition. The design emphasizes usability, enabling end users to drill down into specific segments and identify patterns efficiently.

Tools Used
Microsoft Excel (Power Query for ETL, Pivot Tables for data modeling, and advanced charting for visualization)

Conclusion
This project demonstrates strong capabilities in data cleaning, transformation, and visualization, delivering actionable insights into employee attrition. It highlights the ability to translate raw HR data into meaningful analytics that support strategic decision-making.
